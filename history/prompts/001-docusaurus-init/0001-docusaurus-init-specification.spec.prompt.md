## Prompt

```text
## Goal
Initialize a Docusaurus 3.9.x documentation site with TypeScript, deployed to GitHub Pages via GitHub Actions.

## User Stories
1. As a developer, I can clone the repo, run `npm install && npm start`, and see the site locally at localhost:3000
2. As a developer, when I push to `main`, GitHub Actions automatically builds and deploys to GitHub Pages
3. As a visitor, I can access the site at `https://<username>.github.io/<repo>/` and see the homepage

## Functional Requirements

### R1: Repository Setup
- Initialize Git repository with `.gitignore` (node_modules, build, .docusaurus, .cache-loader)
- README.md with setup instructions

### R2: Docusaurus Scaffold
- Use: `npx create-docusaurus@latest . classic --typescript`
- Version: @docusaurus/core ^3.9.2, @docusaurus/preset-classic ^3.9.2
- React 18.x (confirmed compatible with Docusaurus 3.x)
- Config file: `docusaurus.config.ts` (ESM with TypeScript)

### R3: Docs-Only Mode Configuration
In docusaurus.config.ts:
- Set `docs.routeBasePath: '/'` (docs serve from root URL)
- Set `blog: false` (disable blog feature)
- Keep `pages: false` or minimal (no custom pages needed initially)
- Set `sidebarPath: './sidebars.ts'`

### R4: Project Structure
[Full structure omitted for brevity - see user input]

### R5: Homepage Configuration
In docusaurus.config.ts:
- title: "Physical AI & Humanoid Robotics"
- tagline: "A comprehensive guide to building intelligent robots"
- url: "https://<username>.github.io"
- baseUrl: "/<repo-name>/"
- organizationName: "<github-username>"
- projectName: "<repo-name>"

### R6: GitHub Actions Workflow
File: `.github/workflows/deploy.yml`
- Trigger: push to `main` branch
- Node.js 20.x

### R7: Sidebar Configuration

- Single sidebar named `tutorialSidebar`
- Initially contains only `intro` document
- Type: autogenerated from docs/ folder structure

## Acceptance Criteria
- [ ] `npm install` completes without errors
- [ ] `npm start` launches dev server at localhost:3000
- [ ] Homepage shows title "Physical AI & Humanoid Robotics"
- [ ] Push to main triggers GitHub Actions workflow
- [ ] GitHub Pages serves the built site
- [ ] No TypeScript errors (`npm run build` passes)
- [ ] Console shows no errors in browser

## Out of Scope (Deferred to Later Specs)
- Actual chapter content (just placeholder)

- Docusaurus 3.9.x uses ESM (`export default`) not CommonJS
```
